Shell A Prototype

A 37.5 m² weathertight home, built from standard EPAL pallets in 10 days, monitored for 12 months. Proof that the system works.

Overview

The Shell A prototype is a laboratory disguised as a house. Every wall has temperature and humidity sensors. Every joint is photographed. Every cost is logged. For 12 months, we will collect continuous data on how pallet construction performs in the Algarve climate — and publish it all openly.

Build Schedule

Weeks 1–4 (Pre-build)

Site + Materials + Team

Confirm site, order all materials, recruit 4–6 build team members, hold safety briefing, submit permit if required.

Day 1

Foundation + Floor

Install ground screws, lay pallet floor grid, level and shim, secure pallets, install OSB subfloor.

Day 2

Wall Panels — Prefabrication

Cut I-joists, assemble wall frames, attach OSB sheathing, install insulation, label panels.

Day 3

Wall Erection

Lift and secure all 4 wall panels, brace, check plumb, connect to floor frame.

Day 4

Roof Structure

Install roof beams, pallet grid, roof sheathing, membrane, and roofing material. Weathertight by end of day.

Day 5

Windows + Doors + Sealing

Install windows and door, seal all joints, install external membrane and cladding. Building is weathertight.

Day 6

Electrics

Run cable, install breaker box, sockets, switches, LED lights. Test all circuits. Power on.

Day 7

Plumbing

Install water tank, run PEX pipe, connect sink and shower, test pressure and drainage.

Day 8

Insulation + Interior

Fill gaps, install internal wall and ceiling lining, floor finish, seal joints. Thermally insulated and finished.

Day 9

Systems + Testing

Install solar (if off-grid), test all electrics and plumbing under load, check for drafts, thermal camera scan.

Day 10

Handover + Monitoring Setup

Complete as-built documentation, install and calibrate 20+ sensors, begin 12-month data logging. Celebrate.

Success Criteria

  • Built within 10 active days — Team of 4–6, basic tools, no specialist contractors.
  • Total cost under €15,000 — Materials only, volunteer labour. Verified with receipts.
  • Weathertight and thermally comfortable — No drafts, no condensation, interior 18–24°C for >90% of hours without heating/cooling.
  • All electrics and plumbing working safely — Tested under load, no leaks, no electrical faults.
  • Sensor data flowing continuously — >95% uptime for all sensors over 12 months.
  • No structural deformation after 3 months — Settlement <5 mm, no racking, no joint failure.
  • Build manual is complete enough to replicate — Stranger can build Shell A using only the manual, photos, and videos.
