# Hestia Foundation

Open-source construction systems for affordable, permanent, off-grid family housing.

## Mission

To design, document, and distribute open-source building systems that enable families to construct durable, energy-independent homes using locally available materials — starting with the EPAL pallet-based Shell A prototype in Lagos, Algarve, Portugal.
