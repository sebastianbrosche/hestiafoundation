# Technology Roadmap — Hestia Foundation

Last updated: 2026-05-29
What comes after Shell A

---

## Philosophy

We don't aim for perfection. We aim for **good enough, cheap enough, fast enough** — then iterate.

Each "Shell" is a generation. Shell A is the proof of concept. Shell B improves on A's weaknesses. Shell C adapts for different climates. Shell D scales to multi-unit.

All designs remain open-source. Each generation teaches the next.

---

## Shell A — Proof of Concept (Q3 2026)

**Status:** Planned
**Size:** 37.5 m² (single module)
**Cost:** €25,000
**Build time:** 10 days
**Climate:** Mediterranean (Algarve, Portugal)
**Foundation:** Ground screws
**Structure:** EPAL pallets + I-joists + OSB sheathing
**Insulation:** Mineral wool (100–150 mm)
**Roof:** Onduline bitumen sheets
**Systems:** Basic electrics, plumbing, optional solar
**Monitoring:** 20+ sensors, 12-month data collection
**Target:** Prove the system works, collect data, publish manual

**Shell A is complete when:**
- [ ] Weathertight and comfortable after 12 months
- [ ] All sensor data published
- [ ] Build manual v1.0 released
- [ ] Structural engineer sign-off obtained
- [ ] Total cost documented and verified

## Shell B — Improved (2027, if funded)

**Goal:** Fix Shell A's weaknesses based on data

**Expected improvements:**
- **Thermal bridges** — Address cold spots identified in thermal camera scans
- **Air tightness** — Add vapour barrier and improve sealing; target 3 ACH50
- **Foundation** — Test screw piles vs. ground screws vs. concrete pads; recommend best for different soils
- **Roof** — Test metal roofing vs. tiles vs. green roof; document lifespan and cost
- **Interior finish** — Develop low-cost, durable interior lining options (lime plaster, clay boards, recycled panels)
- **Water harvesting** — Integrate rainwater collection into roof design
- **Compost toilet** — Design and test off-grid sanitation
- **Greywater system** — Simple reed bed or biofilter for shower/sink water

**Size:** Same (37.5 m²) or slightly larger (50 m²)
**Cost target:** €22,000 (10% reduction through optimisation)
**Build time target:** 8 days (20% faster through prefabrication improvements)

## Shell C — Climate Adaptations (2027–2028)

**Goal:** Adapt the system for different European and global climates

### Cold Climate Version (Nordic, Alpine)
- Thicker insulation (200–250 mm mineral wool or cellulose)
- Double-glazed windows (low-E)
- Air tightness target: 1.5 ACH50
- Passive solar design (south-facing glazing, thermal mass)
- Condensation management (vapour-open but wind-tight membrane)
- Snow load engineering (roof reinforcement)

### Hot-Humid Version (Tropical, Subtropical)
- Elevated floor (1m+ on stilts) for flood protection and airflow
- Wide overhangs (1m+) for shade
- Cross-ventilation design (opposing windows, high/low vents)
- Light-coloured roof for reflectivity
- Mosquito screening integrated into wall design
- Mould-resistant interior finishes

### Arid Version (Desert, Mediterranean inland)
- Thermal mass integration (adobe or stone internal walls)
- Narrow windows (reduce solar gain)
- Courtyard option (traditional cooling)
- Dust-resistant sealing
- Water tank integration (rainwater + storage)

### Coastal Version (High wind, salt air)
- Hurricane straps and reinforcement
- Salt-resistant fasteners (stainless steel)
- Elevated foundation (storm surge)
- Corrosion-resistant cladding
- Wind-resistant roof design (low pitch, secured edges)

## Shell D — Multi-Unit and Community (2028+)

**Goal:** Scale from single-family to community housing

### Duplex / Semi-Detached
- Two 37.5 m² units sharing a wall
- Shared foundation, separate systems
- Cost per unit: €20,000 (shared wall and foundation reduce cost)

### Row Housing (4–6 units)
- Shared walls between units
- Shared infrastructure (electrical backbone, water main)
- Cost per unit: €18,000
- Community courtyard / shared garden

### Cluster / Village
- 10–20 units arranged around central space
- Shared amenities: kitchen, laundry, workshop, garden
- Mixed sizes: some 25 m² (single), some 50 m² (family)
- Community build: 20 people, 3 weeks

### Vertical (2-storey)
- Ground floor: living, kitchen, bathroom
- First floor: bedroom, storage
- Strengthened floor joists and wall panels
- Cost: €35,000 for 50 m² (vs. €25K for 37.5 m² single-storey)

## System Improvements (Parallel Tracks)

### Materials Research
- [ ] Test alternative pallet types (CP, national standards)
- [ ] Test pallet alternatives: plastic pallets, composite pallets
- [ ] Bio-based insulation: hemp, straw, mycelium, cork
- [ ] Recycled denim / textile insulation
- [ ] Phase-change materials for thermal mass

### Structural Innovation
- [ ] Pallet arch / vault structures (no I-joists needed)
- [ ] Pallet dome / geodesic adaptations
- [ ] Hybrid systems: pallet + bamboo, pallet + steel frame
- [ ] Seismic-resistant connections (tested on shake table)

### Digital Tools
- [ ] FreeCAD parametric model (adjust dimensions automatically)
- [ ] Web-based cost calculator (input location, get local prices)
- [ ] AR build assistant (phone app showing where to place next pallet)
- [ ] Sensor dashboard (real-time data visualisation)
- [ ] BIM integration (for professional architects)

### Manufacturing / Prefabrication
- [ ] Flat-pack panel factory (local workshop, not centralised)
- [ ] CNC-cut pallet modifications (standardised joints)
- [ ] Pre-insulated panels (factory-applied, faster on-site)
- [ ] Kit delivery: all materials + manual delivered to site

## Research Agenda

### Immediate (2026–2027)
1. Structural performance under Eurocode loads
2. Thermal behaviour in Mediterranean climate
3. Moisture management and condensation risk
4. Durability of heat-treated pallets in outdoor exposure
5. Fire resistance of pallet assemblies

### Medium-Term (2027–2028)
6. Life cycle assessment (LCA) — embodied carbon, energy payback
7. Acoustic performance (STC rating)
8. Seismic performance (shake table testing)
9. Long-term creep and settlement (3–5 year data)
10. Social acceptance and resident satisfaction surveys

### Long-Term (2028+)
11. Automated design optimisation (AI-generated layouts for different constraints)
12. Robot-assisted assembly (pallet placement, screw driving)
13. Biomimetic adaptations (termite mound ventilation, etc.)
14. Policy impact assessment (how many homes built, cost saved)
15. Replication studies in 5+ countries

## Decision Gates

Before proceeding to each generation, we must answer:

### Shell A → Shell B
- [ ] Did Shell A survive 12 months without structural failure?
- [ ] Is interior temperature comfortable for >90% of hours?
- [ ] Is total cost verifiably under €25,000?
- [ ] Did we publish all data?
- [ ] Do we have funding for Shell B?

### Shell B → Shell C
- [ ] Did Shell B show measurable improvement over A?
- [ ] Did we test in at least one additional climate?
- [ ] Is the build manual clear enough for a stranger to follow?
- [ ] Do we have partner demand in other regions?
- [ ] Do we have funding for climate adaptations?

### Shell C → Shell D
- [ ] Do we have proven performance in 3+ climates?
- [ ] Is there demand for multi-unit/community versions?
- [ ] Do we have a builder network trained in the system?
- [ ] Is the system standardised enough for kit production?
- [ ] Do we have funding for scaling?
