# NEB Boost — Application Framework

**Call:** NEB Boost (New European Bauhaus)  
**Amount:** €30,000 per project (20 projects awarded)  
**Target:** Rural settings or communities under 20,000 residents  
**Focus:** Built environment — construction, renovation, adaptation  
**Priorities:** Circularity, carbon neutrality, affordable housing  
**Requirement:** Completed prototype with documentation  
**Lagos:** Qualifies (population ~12,000)

---

## 1. Project Summary

**Title:** The €50,000 Family Home — An Open-Source, Self-Build Housing System for Rural Portugal

**Abstract:**  
The Hestia Foundation will build and document a complete, weathertight, off-grid family home in Lagos, Algarve, using the Ribbed Composite Panel (RCP) — an open-source construction system made from CE-marked timber I-joists, EPAL Euro pallets, and OSB sheathing. The 37.5m² prototype costs €25,000 in materials, requires no concrete or steel, and can be assembled by non-specialists in 5–10 days. The project demonstrates that affordable, beautiful, carbon-negative housing is possible in rural communities without mortgages, specialist labour, or carbon-intensive materials.

**NEB Values:**
- **Sustainable:** Zero concrete, zero steel, upcycled industrial pallets, carbon-negative timber
- **Inclusive:** Non-specialist buildable, mortgage-free, accessible to median-income families
- **Beautiful:** Interior finishes (walnut, bronze, pine tar, engineered oak) that exceed conventional affordable housing standards

## 2. Project Description

### 2.1 What We Will Build
A 37.5m² (7.5m × 5m) single-storey dwelling with:
- RCP wall and roof system (140mm thick, highly insulated)
- Earth screw foundation (no concrete)
- Off-grid systems: 3.5kW solar PV, 10kWh battery, solar hot water, composting toilet
- Interior: engineered oak flooring, walnut worktops, bronze hardware
- Cost: €25,000 materials + €13,180 services = €38,180 total

### 2.2 Where
Lagos, Algarve, Portugal — a rural municipality of ~12,000 residents facing housing affordability crisis (median property price €3,203/m², unaffordable for median earners).

### 2.3 Who Will Build
5–10 volunteer builders led by the Hestia Foundation team. No specialist construction skills required. Build time: 5–10 days weathertight, 10–15 days complete.

### 2.4 Documentation
- Daily photo and video documentation
- Step-by-step build protocol published open-source (CC BY-SA 4.0)
- 12-month sensor monitoring (temperature, humidity, energy, structural)
- Public data dashboard
- Workshop curriculum for future replication

## 3. Why This Fits NEB Boost

### 3.1 Rural Focus
Lagos (population ~12,000) qualifies as a community under 20,000 residents. The Algarve faces severe housing pressure from tourism, pushing residents out. This project provides a locally appropriate, climatically optimised solution.

### 3.2 Built Environment
The project is literally construction — building a home from first principles using an innovative structural system.

### 3.3 Circularity
- EPAL Euro pallets diverted from landfill/incineration
- Timber sequesters CO₂ (every home is a carbon bank)
- Demountable design — entire structure can be disassembled and moved
- No concrete = avoids 5% of global GHG emissions from cement

### 3.4 Carbon Neutrality
- Embodied carbon: ~50% lower than conventional masonry
- Operational carbon: Near-zero (off-grid, passive cooling)
- End-of-life: 100% recyclable or biodegradable materials

### 3.5 Affordable Housing
- Total cost €38,180 — vs. €150,000+ for conventional construction
- No mortgage required — families can save and build outright
- Non-specialist buildable — eliminates labour cost inflation
- Scalable: 500M pallets in circulation = 5M homes without manufacturing anything new
