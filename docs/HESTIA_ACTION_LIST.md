## Step 1: Confirm Founding Members (Week 1)
- [ ] Identify 3 founding members with Portuguese NIFs
- [ ] Draft and sign founder agreement
- [ ] Choose legal form: Associação Sem Fins Lucrativos (recommended)

## Step 2: Register the Foundation (Week 1–2)
- [ ] Draft statutes (estatutos) with open-source clause and asset lock
- [ ] Register with Conservatória do Registo Comercial or via Empresa Online
- [ ] Obtain NIPC (tax number)
- [ ] Open bank account

## Step 3: Secure Prototype Site (Week 2–4)
- [ ] Identify potential land parcel (200m²+, flat, vehicle access, Lagos area)
- [ ] Contact landowners or Municipality of Lagos
- [ ] Negotiate purchase, lease, or municipal site access
- [ ] Conduct soil test for earth screw suitability

## Step 4: Send Partnership Outreach (Week 1–2)
- [ ] Email University of Algarve (civil engineering faculty)
- [ ] Email LNEC (structural testing partnership)
- [ ] Email Lagos Municipality (site + UIA partnership)
- [ ] Email 3 structural engineers (Ordem dos Engenheiros members)
- [ ] Email IHRU (policy alignment)

- [ ] Download full call text from EU Funding & Tenders Portal
- [ ] Identify all required partners (need 3+ entities from 3+ EU countries for Horizon)
- [ ] Draft abstract (max 2 pages)
- [ ] Draft work programme and Gantt chart
- [ ] Begin partner negotiations (university + international partners)
- [ ] **Deadline: 01/12/2026**

## Step 6: Prepare NEB Boost Application (Week 4–8)
- [ ] Build prototype (prerequisite for application)
- [ ] Document build with photos, videos, and cost data
- [ ] Draft NEB Boost application (format to be confirmed when call opens)
- [ ] Submit as soon as prototype is complete and documented

## Step 7: Procure Materials (Week 3–6)
- [ ] Order EPAL pallets (verify heat-treatment certification)
- [ ] Order OSB sheathing, I-joists, earth screws
- [ ] Order off-grid package (solar, battery, hot water)

- [ ] Arrange flatbed trailer delivery

## Step 8: Engage Grant Professional (Week 2–4)
- [ ] Identify 3 candidates with NEB/Horizon/LIFE track record
- [ ] Request case studies and references
- [ ] Interview and select
- [ ] Sign engagement letter
- [ ] Brief on all active applications

## Step 9: Set Up Documentation Infrastructure (Week 1–2)
- [ ] Create GitHub repository (public, open-source)
- [ ] Set up photo/video documentation protocol
- [ ] Configure sensor network and data logger
- [ ] Create public dashboard for live monitoring data
- [ ] Set up social media (LinkedIn, Twitter/X)

## Step 10: Plan Build Workshop (Week 4–8)
- [ ] Recruit 5–10 volunteer builders
- [ ] Confirm 10-day build schedule
