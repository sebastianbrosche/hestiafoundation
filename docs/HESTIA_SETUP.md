## Track A: Foundation Legal Entity

### Step 1: Register Portuguese Non-Profit
- [ ] Choose legal form: Associação Sem Fins Lucrativos (recommended) or Fundação
- [ ] Draft statutes (estatutos) — must include:
  - Non-profit purpose (affordable housing research/dissemination)
  - Open-source commitment
  - Asset lock (património afetado)
  - Governance structure
- [ ] Minimum 3 founding members (need names/NIFs)
- [ ] Register with Conservatória do Registo Comercial (or online via Empresa Online)
- [ ] Obtain NIPC (tax number)
- [ ] Register with DGAE (Direção-Geral das Atividades Económicas)
- [ ] Open bank account
- [ ] Register for VAT (6% reduced rate for affordable housing)
- [ ] Apply for Social Security exemption (if applicable)

### Step 2: Set Up Operations
- [ ] Create email addresses (info@, grants@, research@)
- [ ] Set up accounting system (simple Excel or PME-level)
- [ ] Create document templates (invoices, receipts, contracts)
- [ ] Set up project management system (GitHub Projects or Notion)
- [ ] Create social media accounts (LinkedIn, Twitter/X)

## Track B: Prototype Build

### Phase 1: Site Preparation (Month 1)
- [ ] Secure land (purchase or long-term lease)
- [ ] Conduct soil test for earth screw suitability
- [ ] Obtain temporary structure permit (or full planning permission)
- [ ] Clear and level site
- [ ] Install base camp (container or temporary shelter)
- [ ] Arrange water access (connection or borehole)
- [ ] Arrange electrical connection (for testing grid vs. off-grid)

### Phase 2: Material Procurement (Month 2)
- [ ] Finalise bill of materials from thesis
- [ ] Order EPAL pallets (verify heat-treatment certification)
- [ ] Order OSB sheathing (12mm, certified)
- [ ] Order I-joists (STEICO or equivalent, CE-marked)
- [ ] Order earth screws (calculate load requirements)
- [ ] Order windows and doors
- [ ] Order off-grid package (solar, battery, hot water, composting toilet)
- [ ] Order insulation (rockwool or hempcrete)
- [ ] Order fixings, adhesives, tools
- [ ] Schedule Leroy Merlin Portimão cutting service
- [ ] Arrange flatbed trailer delivery

### Phase 3: Build (Month 3, Days 1–10)
- [ ] Day 1: Foundation (earth screws, timber joists, OSB floor)
- [ ] Day 2–3: Wall panel fabrication + erection
- [ ] Day 4–5: Roof structure (I-joists, OSB deck, weatherproofing)
- [ ] Day 6–7: Insulation + services (electrics, plumbing, solar)
- [ ] Day 8–10: Interior finishing
- [ ] Day 10+: Weathertight + livable

### Phase 4: Monitoring (Months 4–15)
- [ ] Install sensor network (temperature, humidity, electrical, water)
- [ ] Configure data logger + public dashboard
- [ ] Monthly photo documentation
- [ ] Seasonal performance review
- [ ] Structural deflection measurement at 1, 3, 6, 12 months
- [ ] Thermal comfort survey (occupant feedback)
